# Phase 4: 全系统整合与自演化 (System Integration & Self-Evolution)

在前三个阶段，我们已经构建了核心架构、多模态感知和快速习惯通路。第四阶段的目标是提升系统的鲁棒性、异步处理能力以及知识图谱的深度演化，使其真正具备“自生长”的特性。

## Goal
1.  **全异步管道**：消除主循环中的阻塞调用（特别是 LLM 推理和模型加载），提升界面响应速度。
2.  **精密知识图谱**：将 `KnowledgeBase` 从简单的规则列表升级为真正的图结构（Node-Edge），支持多跳查询和规则关联。
3.  **自反馈学习**：引入一个简单的“强化学习”反馈循环，让系统能够根据用户反馈（如“你说得不对”）自动撤销海马体中错误的巩固规则或基底节中的习惯。
4.  **性能监控与日志清理**：优化并发日志输出，解决多线程/多协程下日志交错混乱的问题。

## Proposed Changes

### `neurocortex/core/thalamus.py` & `main.py`
- [MODIFY] **异步模型加载**：将 `_ensure_model` 改为完全非阻塞，避免第一次对话时产生明显的卡顿。
- [MODIFY] **并发推理**：在慢速通路中，允许并行进行海马体检索和知识库查询，缩短端到端延迟。

### `neurocortex/memory/knowledge_base.py`
- [MODIFY] **图结构化升级**：引入 `NetworkX` 或类似的轻量级逻辑，支持规则之间的“包含”、“冲突”或“强化”关系。
- [NEW] **自愈逻辑**：当检测到新提取的规则与现有高置信度规则冲突时，触发额叶进行“认知失调”处理，决定保留哪一条。

### `neurocortex/memory/consolidation.py`
- [MODIFY] **高阶蒸馏**：不仅仅提取单条 if-then 规则，还要分析规则之间的逻辑链，自动构建知识图谱中的边。

### `neurocortex/ui/config_dashboard.py`
- [MODIFY] **知识图谱可视化**：在 Streamlit 仪表盘中增加一个简单的图谱展示页面，直观看到系统学到了什么。
